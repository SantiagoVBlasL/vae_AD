# configs/runs/

Manifiestos de ejecución (comandos exactos) para reproducir entrenamientos/inferencias.

Convención sugerida:
- adni_YYYYMMDD_run.txt
- covid_YYYYMMDD_inference.txt

Incluí:
- comando completo
- git commit hash
- nombre del entorno conda (ej: vae_ad)
- GPU / driver / CUDA si aplica
